# 500円で地球の天気予報をしてみた！ 〜天気予報の科学とAI技術の最前線〜

スーパーコンピュータがないと無理だった天気予報が、今やワンコインでできる時代に！

この動画では、従来の数値天気予報の仕組みから、GoogleやNVIDIAが開発した最新のAI天気予報モデルまで、天気予報の科学とAI技術の最前線をわかりやすく解説します。

後半では、実際にNVIDIAのAI天気予報モデル「FourCastNet」をクラウドGPUで動かして、2025年三が日の天気を予測してみました。かかったコストはたったの500円！

数値計算の基礎から最新AI技術まで、天気予報の裏側を一緒に覗いてみましょう。

🔍 この動画でわかること：
・ 従来の天気予報（数値気象予報）の仕組み
・ 数値計算の基礎（ルンゲ・クッタ法）
・ なぜAIが天気予報に革命をもたらしたのか
・ AI天気予報モデルの速度と精度
・ AIとスーパーコンピュータの協力関係
・ 実際にAI天気予報を動かす方法
